# WorkClock Playful Redesign Mockups

Neue V2-Mockup-Strecke fuer WorkClock mit hellerem, bunterem und verspielterem Look.

## Enthalten

- Mobile Dashboard mit Restzeit-Fokus, Feierabend-Prognose und Primaeraktionen
- Kalender-/Wochenansicht mit Tagesstatus, Wochenkonto und schnellen Vorgaben
- Apple-Watch-Interface mit Countdown, Pause-/Beenden-Aktionen und Komplikationen
- Farb- und Formensprache als Grundlage fuer ein spaeteres Compose Theme

## Datei

`index.html` kann direkt im Browser geoeffnet werden.
